# 🔍 Deepfake Image Detector

This project uses a Convolutional Neural Network (CNN) and Transfer Learning (MobileNetV2) to detect whether an image is a real photograph or an AI-generated deepfake.

## 🛠️ Tech Stack
- **Language:** Python 3.11
- **Framework:** TensorFlow / Keras
- **Frontend:** Streamlit
- **Model:** MobileNetV2 (Transfer Learning)

## 🚀 How to Run
1. Clone this repository.
2. Create a virtual environment: `python3 -m venv venv`
3. Activate it: `source venv/bin/activate`
4. Install dependencies: `pip install -r requirements.txt`
5. Train the model: `python3 train_model.py`
6. Run the app: `streamlit run app.py`
